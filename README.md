## 简介

这是一个利用 `github actions` 实现的定时任务程序。

## github actions secrets

| 变量               | 备注                                  |
| ------------------ | ------------------------------------- |
| EMAIL_FROM         | email 发送者，ex: "昵称" <xxx@qq.com> |
| EMAIL_TO           | 接受邮件的 email                      |
| EMAIL_USER         | 邮箱服务的用户名                      |
| EMAIL_PASS         | 邮箱服务的密码                        |
| JUEJIN_COOKIE_JSON | COOKIE JSON                           |
